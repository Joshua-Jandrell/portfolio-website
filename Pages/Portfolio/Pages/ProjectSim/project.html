Project

A text-based resource management project simulator.

You are a varsity student and you have just got your first project of the semester: make a game with twine. Manage your energy, focus, and stress levels to ensure you submit quality work on time. (Yes, this game is semi-autobiographical.)

Project is a resource management game in which the levels of resources effect the appearance of the environment. Getting too tired, stressed, or losing focus will make it harder for you to play the game.

I learnt a lot in the three weeks spent working on this game. It was my first-time using twine and css. It found it to be a changing but fulfilling experience.

Now good luck getting that A!

Play

  • Play in browser!

Features include:

  • Exactly 5 minute of gameplay.
  • Variables effected by incremental decisions
  • Cat laucnhing using the unity physics engine (and Hooke's Law)
  • CSS styling effected by variables
  • Decision effected by CSS in a vicious feedback loop? - you tell me.
